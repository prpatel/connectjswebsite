Ionic 2 workshop: Angular for native mobile appsMike Hartington

Hi there! For this Ionic workshop, we're going to want to make sure we have a few things installed.

- Node
  We're going to want to have NodeJS 8.x installed and set in our path.
  Visit https://nodejs.org/ to install it and npm, and verify it has added
  by running `node -v` and `npm -v`

- Git
  We're also going to want to make sure we have git installed ad well.
  Git (and git-bash for windows) will be needed for a small part of the workshop.
  Download Git from https://git-scm.com and complete the installation.

- Cordova CLI
  Now that node and npm are installed, install the cordova CLI with `npm install -g cordova`.
  If you're on a windows machine, you'll want to use an admin command prompt. For mac or linux,
  you may need to use `sudo`


- Ionic CLI
  Similar to Cordova CLI, we'll need to install the ionic CLI. Please run `npm install -g ionic`
  Same notes for Windows and Mac/Linux.

- Ionic Pro account and Ionic View
  We'll be using the free tier of Ionic Pro in order to deploy our app to a device.
  Please visit https://dashboard.ionicjs.com/signup and sign up for an account,
  and download "Ionic View" From the iOS App Store or Google Play.


- Editor with Typescript support
  Ionic is written with typescript, a superset of JavaScript. In order to get the best
  experience, it's best to install any editor plugin that adds typescript support.

  Atom: atom-typescript
  Sublime: Sublime TypeScript
  VScode: Support out of the box

  If you're not sure what one is right, I suggest VSCode, and it provides the best
  "out of the box" experience.


That's it! See you all in a few weeks and feel free to get in touch with me directly
if you run into any issues for the set.

mike@ionic.io, "ConnectTech Workshop setup"
