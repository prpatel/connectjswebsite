Animating the User Interface: Designing for MotionAngie Terrell

This workshop will include mostly hands-on exercises on how to prototype animation into your designs. We will be using primarily Principle, Flinto for Mac, and Keynote. Please make sure you have these tools before coming. If you don’t already own Principle or Flinto, there is a 14-day trial, which you can download prior to the workshop. By the end of the workshop, you will definitely have a clear idea as to whether you would want to purchase either one.

ES6 (and beyond) WorkshopKent C Dodds

Thanks for signing up to come to my workshop. I have a few things that I'd like everyone to do. Please follow these instructions as it will ensure we have as few problems as possible with the set up and we'll be able to spend as much time as possible with the workshop!
                                          
Go to this repo and follow the setup instructions: https://github.com/kentcdodds/es6-workshop (est. time: 3 mins)

Go to this repo and follow the Frontend Masters setup instructions: https://github.com/kentcdodds/es6-todomvc (est. time: 3 mins)

Fill out this form: http://kcd.im/connect-es6 (est. time: 2 mins)

Just a few notes about the workshop:
In the morning (and maybe part of the afternoon) we'll go over ES6 features using the first workshop
In the afternoon, we'll go through as much as we can of the ES6 TodoMVC workshop which will demonstrate the tooling you'll need to transpile ES6 for running in browsers you're shipping your apps to today. 
This workshop assumes a fundamental understanding of JavaScript. I'll assume that you're a beginner/inexperienced with ES6.
